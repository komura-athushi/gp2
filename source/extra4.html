外伝四章 k2SLEditor

    この章で学ぶこと

    • k2SLEditor

  • Ex4.1 k2SLEditorとは

    • k2Engineを用いてのゲーム開発における画像配置のサポート(ゲームをいちいち実行しなくても、画像がどのように表示されるかわかります)や、画像ファイルのdds変換をするツールである。

  • Ex4.2 説明

    1. ツールの起動
      2. tools/k2SLEditor/k2SLEditor.exeからツールを起動。
    2. 画面説明
      3. 大まかなツールの説明
    3. 使い方
      4. 画像ファイルを読み込んで、配置します。
      k2Engineでのゲーム制作で、画像詳細欄の通りに値を設定すると、k2SLEditorと同じように画像が表示されます。
    4. 画像の詳細
    5. dds変換
      6. 「ファイル」→「レベル書き出し」を選択。 適当なフォルダを選んで保存すると、ddsファイルが出力されている。


  • Ex4.3 開発環境

    • 開発環境
      • Windows10

    • 使用言語
      • Python

      • toolsProj/k2SLEditorProjにPythonのソースコードが入ってます。
    • 製作期間
